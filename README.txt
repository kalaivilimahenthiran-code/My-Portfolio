KALAIVIZHI MAHENDRAN - PORTFOLIO WEBSITE
==========================================

This is a modern portfolio website built with Next.js, React, and Tailwind CSS.
It features smooth animations, a dark theme, and showcases projects, skills, and experience.


PREREQUISITES
-------------
Before running this project, make sure you have installed:
- Node.js (version 18 or higher recommended)
- npm (comes with Node.js installation)

To check if you have Node.js and npm installed:
- Open terminal/command prompt
- Type: node --version
- Type: npm --version


INSTALLATION
------------
1. Extract the zip file to your desired location
2. Navigate to the project folder using terminal/command prompt
3. Run the following command to install all dependencies:

   npm install

This will install all required packages including:
- Next.js (React framework)
- React and React DOM
- Framer Motion (animations)
- React Icons
- React Hot Toast (notifications)
- Tailwind CSS (styling)
- And other dependencies


RUNNING THE PROJECT
------------------
After installation, you can run the development server:

   npm run dev

The website will be available at: http://localhost:3000

Open your web browser and navigate to http://localhost:3000 to view the portfolio.


BUILDING FOR PRODUCTION
------------------------
To create an optimized production build:

   npm run build

After building, you can start the production server:

   npm start

The production build will be faster and optimized for deployment.


PROJECT STRUCTURE
-----------------
- src/app/          - Next.js app directory (pages, layout)
- src/components/   - React components (public and admin)
- src/data/         - Data files (projects, skills, experience, settings)
- src/context/      - React context providers
- public/           - Static assets (images, CV, etc.)
- package.json      - Project dependencies and scripts


CUSTOMIZATION
-------------
To customize the portfolio content, edit the following files in src/data/:
- settings.js      - Personal information, contact details, social links
- projects.js      - Project portfolio
- skills.js        - Technical skills
- experience.js    - Work experience
- gallery.js       - Gallery images


TROUBLESHOOTING
---------------
If you encounter any issues:

1. Make sure Node.js is installed (version 18+)
2. Delete node_modules folder and package-lock.json
3. Run npm install again
4. Clear browser cache if changes don't appear

For more information about Next.js, visit: https://nextjs.org/docs
